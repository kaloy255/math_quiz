# Badge Images

This folder contains the badge images for the student achievement system.

## Required Images

Please add the following PNG images (48x48 pixels recommended):

1. **rookie.png** - For "Rookie" badge
2. **learner.png** - For "Learner" badge
3. **explorer.png** - For "Explorer" badge
4. **challenger.png** - For "Challenger" badge
5. **solver.png** - For "Solver" badge
6. **master.png** - For "Master" badge
7. **legend.png** - For "Legend" badge

## Design Guidelines

- **Size**: 48x48 pixels (or any square aspect ratio)
- **Format**: PNG with transparency
- **Icon**: Black starburst/badge icon on transparent background
- **Background**: The app will apply a black container background

## Current Status

Currently, the app will display a fallback star icon if the images are not found.
